How to install?

Install on Windows

  1. Make sure you have at least Java 8 installed
  2. Download your chosen Quail version
  3. Place qre.jar and/or qdk.jar into some folder on your computer

Install on Linux

  1. Make sure you have at least Java 8 installed
  2. Download your chosen Quail version
  3. Place qre.jar and/or qdk.jar into some folder on your computer
